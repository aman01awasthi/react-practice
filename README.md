# React Practice

Learning React as part of a 60-day fullstack roadmap.

## Tech Stack
- React (Create React App)
- JSX
- Component-based architecture

## Concepts Covered
- Functional components
- Props
- JSX syntax
- Component composition

## Components Built
- Header — displays app title via props
- UserCard — displays name and role via props
- Button — displays label via props
- ProfileCard — displays name, role, bio via props — reused 3 times with different data

## How to Run
```bash
npm install
npm start
```
